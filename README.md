#  Piano Keyboard

An interactive piano keyboard web application built with HTML, CSS, and JavaScript. Play musical notes by clicking on the keys with your mouse or using your computer keyboard.

##  Features

-  Interactive visual piano keyboard interface
-  Mouse click support to play notes
-  Keyboard input support for playing
-  Real-time audio playback for white keys
-  Pure vanilla JavaScript (no frameworks)
-  Responsive design

##  How to Use

1. **Clone the repository**
```bash
   git clone https://github.com/sohamghadai42/Piano.git
```

2. **Navigate to the project directory**
```bash
   cd Piano
```

3. **Open in browser**
```bash
   open index.html
```
   Or simply double-click the `index.html` file

4. **Play the piano**
   - Click on the white keys with your mouse
   - Use your keyboard keys to play notes
